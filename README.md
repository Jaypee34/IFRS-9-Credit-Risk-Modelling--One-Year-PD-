# IFRS-9-Credit-Risk-Modelling--One-Year-PD-
One-year PD model for retail credit built using logistic regression under IFRS 9. Uses key risk drivers including utilisation, arrears, income, bureau score, employment, and delinquency. Achieves strong performance (AUC 0.92, Gini 0.84, KS 0.71) with stable cross-validation. Suitable for ECL, SICR, and credit risk monitoring.
